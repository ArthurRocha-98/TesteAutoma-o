# TesteAutomacao
Teste automatizado


# Automação Cypress + Cucumber

## Tecnologias

- JavaScript
- Cypress
- Cucumber (BDD)

## Instalação

```bash
npm install
```

## Executar testes

Modo Headless

```bash
npx cypress run
```

Modo Interface

```bash
npx cypress open
```

## Estrutura

- Testes WEB
- Testes API
- Page Objects
- Step Definitions
- Features BDD
